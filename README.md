# Numerical Solver

A full-stack numerical methods solver designed to solve mathematical and engineering problems using multiple numerical analysis algorithms.
The project provides an interactive and user-friendly interface for experimenting with different root-finding and numerical computation methods.

---

# Features

* Bisection Method
* False Position Method
* Newton-Raphson Method
* Secant Method
* Gaussian Elimination
* Interactive User Interface
* Fast Numerical Computation
* Dynamic Input Handling
* Real-Time Results Display

# Installation & Usage

## Clone the Repository

```bash id="lx6a8w"
git clone https://github.com/MohamedSami01/numerical-project.git
```

---

## Open the Project

```bash id="y9k7xe"
cd numerical-project
```

---

## Run the Frontend

Open `index.html` in your browser.

---

## Compile the Backend

```bash id="i0ww7h"
g++ main.cpp -o solver
```

---

## Run the Program

```bash id="0b0q7g"
./solver
```
